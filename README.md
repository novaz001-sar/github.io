# Spatial Cognition Game

Static GitHub Pages build for the spatial cognition training game.

## Structure

- `index.html` - main game shell and GitHub Pages entry point.
- `game16.2.html` - legacy redirect kept so old links still work.
- `css/game.css` - extracted game UI and overlay styles.
- `js/intro.js` - introduction video timing behavior.
- `js/game.js` - main Three.js game logic.
- `assets/video/intro.mp4` - introduction video asset.

## Deployment

This repository is deployed by GitHub Pages from the `main` branch. Pushes to `main` update the live site automatically once GitHub Pages finishes publishing.
